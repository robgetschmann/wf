Implementation module for memory wrapper functions

Provide a wrapper for calloc, malloc, and realloc. The wrapper function calls the C library version. If memory allocation fails the wrapper will invoke and assert() to terminate the process.\hypertarget{wf_8h_LICENSE}{}\subsection{L\+I\+C\+E\+N\+SE}\label{wf_8h_LICENSE}
